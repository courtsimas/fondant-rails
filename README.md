# FondantRails

Use [fondant](https://github.com/ovenbits-ingredients/fondant) in Rails' asset pipeline.

## Installation

Add this line to your application's Gemfile:

    gem 'fondant-rails', '~> 0.2.1', github: 'ovenbits-ingredients/fondant-rails'
